# Linnet Consult Website

Landing page for Linnet Consult — finansautomation for danske virksomheder.

## Stack
- Static HTML/CSS/JS (single file)
- Deployed via Vercel
- No build step required

## Local development
```bash
npx serve public
```

## Deployment
Push to `main` → Vercel deploys automatically.
